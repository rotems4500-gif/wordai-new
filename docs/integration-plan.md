# תוכנית עבודה לפיתוח ממשק Word הוליסטי משולב AI (שלב 2)

מסמך זה מפרט את סדר הפעולות להמרת עיצוב ה-HTML החדש (`draft-ui.html`) לאפליקציית React מלאה על בסיס העורך הקיים (Tiptap), כולל הטמעת פיצ'רי ה-AI השונים שעליהם הוסכם (עט קסמים, תיבת שיח צפה, ודשבורד ניהול).

## שלב 1: פירוק והמרת ה-UI לרכיבי React
מטרת שלב זה היא להחליף את המעטפת הבסיסית שבנינו קודם במעטפת העשירה מה-Draft, מבלי לשבור את פונקציונליות השמירה והיצוא שכבר קיימת.
*   **רכיב ה-Top Bar:** יצירת פאנל עליון כחול הכולל את שם המסמך וחיפוש מדמה.
*   **מודול ה-Ribbon (סרגל הכלים):** 
    *   המרת מבנה הטאבים (בית, הוספה, עיצוב, פריסה וכו') לרכיב משתנה (State-based).
    *   חיווט הכפתורים השונים לפקודות קיימות ב-Tiptap (Bold, Italic, יישור טקסט ורשימות).
*   **פאנל צדדי (Taskpane):** עדכון רכיב ה-`AiSidebar` לעיצוב החדש והלבן ("AI Perfect Assistant") הכולל את ציון העורך, ההערות ופעולות הסוכנים במבנה של כרטיסיות.

## שלב 2: שדרוג חווית העורך (Tiptap as A4)
*   **עיצוב עמוד מתקדם:** התאמת הקלאסים של אזור ה-Tiptap שייראה בדיוק כמו דף A4 טיפוסי כולל אפליקציית צלליות (Box Shadow) ושוליים.
*   **פיצ'ר ספירת מילים:** הוספת רכיב Footer פשוט (Status Bar) המציג את כמות המילים במסמך בזמן אמת, המסתמך על מידע אוטומטי מה-Tiptap.

## שלב 3: אינטראקציות AI תוך-כתיבה (In-Flow AI)
שלב זה מכניס את רעיונות מוצר העתיד לתוך זרימת העבודה במקום טעינת רקע.
*   **עט הקסמים (Magic Pen):** הוספת כפתור צף קטן (Widget) שיופיע בצד ימין (בשוליים) בשורה שבה הסמן נמצא כרגע. לחיצה עליו תפתח תפריט AI קצר הרלוונטי לפסקה הספציפית (למשל: "נסח מחדש", "תקן שגיאות").
*   **חלונית שיח מהירה (Ctrl+J):** האזנה לקיצור דרך במקלדת שיקפיץ תיבת טקסט מרחפת (בסגנון Spotlight) מעל המסמך. תיבה זו תשמש כצ'אט המודע להקשר ותאפשר לבקש דברים כמו "צור כאן טבלה על X" או "סכם את מה שסימנתי".

## שלב 4: דשבורד AI והגדרות אגנוסטיות
הוצאת מערכת ניהול הטוקנים מחוץ לעורך עצמו ויצירת מרכז בקרה מסודר:
*   **מסך עזר (Settings Modal):** יצירת ממשק המעוצב כמו "הגדרות מתקדמות".
*   **ריבוי מנועים (Multi-Model):** הכנה בממשק בחירת מנוע (לדוגמה Gemini לעומת OpenAI או Anthropic) עבור משתמשים בעלי מפתחות API שונים, כך שהמערכת מאפשרת להזין מפתחות אישיים ולהחליף את מנוע הליבה בלחיצה, בלי תלות בחומרה המקומית.
*   **שמירה כטיוטה:** שדרוג השמירה ה"ממשית" למניעת אובדן דאטה אם הדפדפן נסגר, על ידי שמירת LocalStorage הדוקה.

---
**הערה טכנית:** כל קריאת AI תופעל באופן ידני (On-Demand) למניעת בזבוז טוקנים, ותציג חיווי טעינה מלא למשתמש.
