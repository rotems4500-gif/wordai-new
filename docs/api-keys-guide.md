# מדריך למתחילים: מה זה API Key ואיך מקבלים אותו

אם זו הפעם הראשונה שאתה רואה את המושג API Key, הכל טוב.
אפשר לחשוב על זה כמו סיסמה אישית שמאפשרת לתוסף לדבר עם שירות ה-AI עבורך.

## למה צריך שני מפתחות

- Gemini API Key: עבור כתיבה, ניסוח, שכתוב ופעולות הצ'אט הראשיות.
- Perplexity API Key: עבור חיפוש מקורות וסיוע מחקרי.

## איך מוציאים Gemini API Key

1. פתח את הקישור: https://aistudio.google.com/app/apikey
2. התחבר לחשבון Google שלך.
3. לחץ על יצירת מפתח חדש (Create API key).
4. העתק את המפתח ושמור אותו.
5. בתוסף, פתח Settings והדבק בשדה Gemini API Key.

## איך מוציאים Perplexity API Key

1. פתח את הקישור: https://www.perplexity.ai/settings/api
2. התחבר לחשבון Perplexity שלך.
3. צור מפתח חדש.
4. העתק את המפתח ושמור אותו.
5. בתוסף, הדבק בשדה Perplexity API Key.

## טיפים חשובים למתחילים

- לא משתפים API Key עם אף אחד.
- לא מעלים API Keys ל-GitHub.
- אם יש חשש שדלף מפתח, מוחקים אותו ומנפיקים חדש.

## מה עושים אם לא עובד

- ודא שהמפתח הודבק במלואו בלי רווח בתחילה/בסוף.
- ודא שלחצת על "שמור מפתחות".
- סגור ופתח מחדש את חלונית התוסף.
