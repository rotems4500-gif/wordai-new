מדריך למתחילים: מה זה API Key

API Key הוא מפתח אישי שמאפשר לתוסף להתחבר לשירותי AI בשמך. חשוב לשמור עליו פרטי, כמו סיסמה.

למה צריך שני מפתחות

  • Gemini API Key: לשכתוב, ניסוח, כתיבה ופעולות הצ'אט.
  • Perplexity API Key: למחקר, חיפוש מקורות והצלבות.

איך מקבלים Gemini API Key

  1. היכנס לקישור: Google AI Studio - API Keys
  2. התחבר לחשבון Google.
  3. צור מפתח חדש והעתק אותו.
  4. בתוסף, עבור ל-Settings והדבק בשדה Gemini API Key.

איך מקבלים Perplexity API Key

  1. היכנס לקישור: Perplexity API Settings
  2. התחבר לחשבון Perplexity.
  3. צור מפתח חדש והעתק אותו.
  4. בתוסף, הדבק בשדה Perplexity API Key.

אם משהו לא עובד

  • בדוק שהמפתח הודבק בלי רווחים בתחילה/בסוף.
  • לחץ על "שמור מפתחות".
  • סגור ופתח מחדש את חלונית התוסף.
